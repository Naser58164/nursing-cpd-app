## Column Descriptions

| Column | Purpose | Valid Values | Example |
|--------|---------|--------------|---------|
| **A: ID** | Unique identifier | ANN001, ANN002, ANN003... | ANN001 |
| **B: Title** | Short announcement title | Any text (50 chars max) | "CPD Deadline" |
| **C: Message** | Full announcement text | Any text (500 chars max) | "Complete 20 hours..." |
| **D: Priority** | Importance level | High, Medium, Normal, Low | High |
| **E: Expiry Date** | When to stop showing | Future date (YYYY-MM-DD) | 2024-12-31 |
| **F: Status** | Active or not | Active, Inactive | Active |
| **G: Created Date** | When created | Date or =TODAY() | 2024-11-12 |
| **H: Created By** | Author name | Any text | Admin |

## Quick Setup Steps

### 1. Create Sheet
- Click "+" at bottom of Google Sheets
- Name it: **Announcements** (exact spelling)

### 2. Add Headers
- Copy Row 1 above
- Paste into A1
- Data will split into columns automatically

### 3. Format Headers
- Select Row 1
- Make bold
- Add background color (light blue)
- Freeze row: View ‚Üí Freeze ‚Üí 1 row

### 4. Add Data Validation

**For Priority (Column D):**
- Select D2:D1000
- Data ‚Üí Data validation
- List: `High,Medium,Normal,Low`
- Save

**For Status (Column F):**
- Select F2:F1000
- Data ‚Üí Data validation
- List: `Active,Inactive`
- Save

### 5. Add Sample Data
- Copy Rows 2-6 above
- Paste into A2

### 6. Save & Test

- Deploy Code.gs
